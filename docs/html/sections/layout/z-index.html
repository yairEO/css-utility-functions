--z()

Semantic z-index scale to avoid magic numbers and conflicts. Returns layer × 100. Layer reference: 1=dropdown, 2=sticky, 3=fixed, 4=drawer, 5=modal, 6=popover, 7=tooltip, 8=toast, 9=max.

Parameters

  • --layer (required) — Layer number (1-9).

Basic Usage

.dropdown { z-index: --z(1); } /* 100 */ .header { z-index: --z(3); } /* 300 */ .modal { z-index: --z(5); } /* 500 */ .tooltip { z-index: --z(7); } /* 700 */
